What is Internet?

The Internet is a global network of interconnected computers and devices that communicate with each other.

Example: opeing youtube in the laptop

What is Browser?

A software application used to access and view website on the internet

Example: Chrome,Firefox,Edge

What is URL?

Uniform Resource Locator is the address used to access a specific resource on the internet

Example: htt://www.google.com

What is HTTP?

HyperText Transfer Protocol is a protocol used to transfer data between a web browser and a website without encryption

Example: http://example.com (not secure)

What is htpps:

HyperText Transfer Protocol Secure is a secure version o HTTP that encrypts data between your browser and the website to protect it.

Example:https://www.amazon.com (ensures safe online shopping)

What is Website and Webapp?

A website is a collection of interconnected web pages accessible through the internet, typically identified by a domain name

Example: www.google.com

A web app is an application that runs in a web browser, allowing users to interact with it over the internet.

Example: Google Docs

What is Domain and division of domain?

A Domain is the human-readable address of a website used to access it on the internet

There are mainly divided into 3 types

Subdomain

Main Domain

Top level Domain

Example: www.google.com

www is a Subdomain

google is a main domain

.com is a top level domain

What is IP address?

Internet Protocol address is a unique set of numbers assigned to each device connected to the internet to identify and communicate with other devices.

Example: 192.168.1.1

What is the difference between Markup language and Programming Language?

Programing language is used to write insructions that perform computing and logic

Example: Python,C++

Markup language is used to structure and format content on the web without performing any logic

Example: HTML
